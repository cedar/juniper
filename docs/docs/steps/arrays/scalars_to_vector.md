# ScalarsToVector

```python
ScalarsToVector(name: str, N_scalars: int)
```

Combines scalar inputs into one vector.

## Slots

| Slot | Description |
|------|-------------|
| Inputs | `in0`, `in1`, ... `in{N-1}` scalar inputs |
| Outputs | `out0` vector of length `N_scalars` |

## Import

```python
from juniper import ScalarsToVector
```
