# VectorToScalars

```python
VectorToScalars(name: str, N_scalars: int)
```

Splits a vector into scalar output slots.

## Slots

| Slot | Description |
|------|-------------|
| Inputs | `in0` vector |
| Outputs | `out0`, `out1`, ... `out{N-1}` scalar outputs |

## Import

```python
from juniper import VectorToScalars
```
