# ReorderAxes

```python
ReorderAxes(name: str, order: tuple)
```

Permutes axes of `in0`.

## Slots

| Slot | Description |
|------|-------------|
| Inputs | `in0` array |
| Outputs | `out0` transposed array |

## Import

```python
from juniper import ReorderAxes
```

## Notes

- `order` is passed as the target axis order.
