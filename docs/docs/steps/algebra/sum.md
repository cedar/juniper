# Sum

```python
Sum(name: str)
```

Sums all incoming values on `in0` componentwise.

## Slots

| Slot | Description |
|------|-------------|
| Inputs | `in0` accepts multiple arrays with compatible shapes |
| Outputs | `out0` sum array |

## Import

```python
from juniper import Sum
```

## Notes

- Use `Sum` to merge branches with the same shape.
