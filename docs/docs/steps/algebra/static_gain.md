# StaticGain

```python
StaticGain(name: str, factor: float)
```

Multiplies `in0` by a constant factor.

## Slots

| Slot | Description |
|------|-------------|
| Inputs | `in0` array |
| Outputs | `out0` array with the same shape as `in0` |

## Import

```python
from juniper import StaticGain
```
