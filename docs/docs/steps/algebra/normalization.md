# Normalization

```python
Normalization(name: str, function: str)
```

## Description
Normalizes incoming step using specified norm function.

## Parameters    
- function : str(InfinityNorm, L1Norm, L2Norm)

## Slots
- Input: jnp.ndarray
- output: jnp.ndarray
