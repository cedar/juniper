# Algebra Steps

Algebra steps perform mathematical operations on input matrices. These are all static steps.

| Step | Description |
|------|-------------|
| [AddConstant](add_constant.md) | Adds a scalar constant to every element |
| [ComponentMultiply](component_multiply.md) | Element-wise multiplication of multiple inputs |
| [Convolution](convolution.md) | Convolves input with a kernel (static or dynamic) |
| [Normalization](normalization.md) | Normalizes input using a specified norm |
| [StaticGain](static_gain.md) | Multiplies input by a scalar factor |
| [Sum](sum.md) | Element-wise sum of multiple inputs |
| [TransferFunction](transfer_function.md) | Applies a sigmoid non-linearity |
